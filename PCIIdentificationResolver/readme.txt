           PCIIdentificationResolver Library
------------------------------------------------------------
PCIIdentificationResolver  is a  library  released under the 
LGPLv3 license,  containing  the list of known  PCI  vendors,
devices and other related information for .Net

For more  information about this  library,  please check out
our GitHub page:
https://github.com/falahati/PCIIdentificationResolver

Please check the `PCIIdentificationDatabase` static methods.
